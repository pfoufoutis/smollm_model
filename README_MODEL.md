# SmolLM2-135M-Instruct (Quantized for Android)

This repository contains a quantized version of the **SmolLM2-135M-Instruct** model, specifically formatted for use with Google MediaPipe on Android devices.

## Model Details
- **Original Model:** [HuggingFaceTB/SmolLM2-135M-Instruct](https://huggingface.co/HuggingFaceTB/SmolLM2-135M-Instruct)
- **Format:** GGUF/Binary (.bin)
- **Quantization:** 4-bit
- **License:** Apache License 2.0

## Usage
This model is intended for local, offline inference in the "Text Santa" Android application to provide magical, age-appropriate festive responses for children.
